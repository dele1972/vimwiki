git - changing a commit message
===============================


Commit has not been pushed online
---------------------------------
1. navigate to the repository folder
2. git commit --amend
3. edit the commit message, and save the commit


Amending the message of the most recently pushed commit
-------------------------------------------------------
4. git push --force
