# MySQL Workbench

## Install

Ubuntu -> snap (Ubuntu Store)

## Configuration

localhost
3306
root@docker

### PW Error

1. Install gnome-keyring

```shell
sudo apt-get install gnome-keyring
```

2. Permission Error

Set Permissions for MySQL Workbench via Ubuntu Software center

- Enable Read, add ...

[Source](https://stackoverflow.com/a/64265457)
